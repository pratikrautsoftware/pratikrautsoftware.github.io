1. INTRODUCTION


Advanced ReTemp Cleaner deletes/empties all the unwanted files and folders located in the Recycle Bin and also in the TEMP Folder.

1.1 Help

There are total three options for the user and each option has a different function.

1. Delete Recycle Bin- Deletes all the files and folders permanently from the RECYCLE BIN ( Only if they are located in the RECYCLE BIN).

2. Delete Temporary Files- Deletes all the temporary files and folders permanently from the TEMP Folder (Only if they are located in the TEMP Folder).

3. Delete ReTemp- Deletes all the files and folders permanently from both the RECYCLE BIN and even from the TEMP Folder (Only if they are located in the RECYCLE BIN and in the TEMP Folder).

1.2 About this software
 
Advanced ReTemp Cleaner deletes/cleans all those unwanted files and folders from the RECYCLE BIN and also from the TEMP Folder, the great advantage of this is that it allocates more free space so that the user can store its data or install programs on the drive ( i.e. OS's main drive).

Resolution - 800x600

Graphic Card - 256 MB

RAM - 2 GB or higher

Resolution - 1024x768

Graphic Card - 512 MB or higher
