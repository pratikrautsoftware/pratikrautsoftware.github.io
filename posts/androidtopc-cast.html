AndroidToPC Cast

Version - 1.0

AndroidToPC Cast casts the connected android device's screen to the host PC using the Android Cast.
User can control the android device from computer too using keyboard keys, mouse or even touch if it has.

Features



- New User friendly GUI Interface

- No need to download and install separate adb

- Install New ADB if users want the latest version

- View connected android devices

- Connect android device to different Android Cast even if one is already connected

- Disconnect Android Cast separately or all at once

FAQ


 
1. Errors during installation of AndroidToPC Cast.
 
Answer 1
 
Make sure that you run 'AndroidToPCCast_setup.exe' as administrator when installing AndroidToPC Cast.
 
 
 
2. Device not showing in devices list or Android device's screen is not getting cast?

Answer 1

If device is not shown in devices list or Android device's screen is not getting cast on Android Cast then make sure that the device's complete drivers including Usb Adb Drivers and Usb Adb Debug Bridge drivers of the respected device are installed.

Answer 2

It is highly recommended that you install AndroidToPC Cast in the directory where it does not require administrator rights

eg: C:\
      Desktop
