TriExGen


 


TriExGen

Version - 1.0

TriExGen - A trial extender generator created by Pratik.Raut generates a crack for any 'trial period' software in '.vbs' extension format.


How it works?

The generated crack acts as a 'parent application' which takes system to the period of the 'Installed date' of the software and then runs the software.


Once user finishes working with the software then again the 'parent application' gets system back to the current period.
