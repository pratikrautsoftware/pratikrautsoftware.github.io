FAQ


1. How to create multiple sticky notes?

Answer

For creating multiple sticky notes, go to the root directory of the 'Sticky Notes' application where the .exe file is located and create multiple instances of the application into the same directory.

eg:  StickyNotes1.exe
  StickyNotes2.exe
  StickyNotes3.exe

After creating these instances, make sure to paste or create their shortcuts into the 'Startup' folder for launching them automatically during startup.


2. Why don't some elements of the palette have any effect?

Answer

Some palette options like the 'Transparency' cannot be applied to some elements or parts of the application, it is therefore for this reason, such options or features still won't have any effect despite their availability and are globally discarded by default even if they are attempted to be applied or used.
