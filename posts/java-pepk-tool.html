Java PEPK Tool






Java PEPK Tool

Version - 1.0

Java PEPK Tool is the GUI version of the PEPK Tool that is created by Pratik Raut

Features



- New User friendly GUI Interface

- Generate Private Keys within a click in less time

You can export the private keys in .pem or .zip format as per your needs.

How to use?



PEPK: Path to the pepk.jar (eg C:\Users\YourName\Downloads\pepk.jar)

Keystore: Path to keystore which you use to sign your release APK.
Could be a file of type *.keystore or *.jks or without extension.
(eg C:\Android\mykeystore or C:\Android\mykeystore.keystore or C:\Android\mykeystore.jks, etc...)

Key Alias: The name of the alias you use to sign the release APK. (eg. key0)

Save Output in: The path of the output file. (eg C:\Android)

Output: Save output file with .pem or .zip extension or Save output as .pem or .zip
(eg C:\Android\private_key.pem or C:\Android\private_key.zip)

Google Encryption Key: This encryption key should be always the same. You can find it in the App Signing page, copy and paste it.

Should be in this form:
eb10fe8f7c7c9df715022017b00c6471f8ba8170b13049a11e6c09ffe305
6a104a3bbe4ac5a955f4ba4fe93fc8cef27558a3eb9d2a529a2092761fb8
33b656cd48b9de6a

Export: Exports/Generates the private key file

Note: While exporting/generating the private key file you might be prompted to enter the 'keystore' and the 'key' password. The console window will not display the password even though you might be entering the passsord but it is entered.

Private Key will have the same name as that of the keystore.
(eg Keystore: key.keystore or key.jks
      Private Key: key.pem or key.zip)
