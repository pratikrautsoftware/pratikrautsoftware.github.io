Android Navbar

Version - 1.6


Description


Do You?

Want a software navigation bar upon hardware navigation bar?

Want to use software navigation bar over oxygen pill or bottom bar?

Want to completely remove the dependency of remote on AndroidTV or AndroidTV box?

Here, it is

Android Navbar - another life and money savior

Extra Options - special hidden options under the hood

Android Navbar viz. Android Navigation Bar generates the navigation bar whilst simulating the actions of a common android navigation bar.


Features

- Supports all android versions through Android 5.1 (Lollipop MR1)
- Comes with Extra Options
- Can change the position of docking
- Has 11 different and unique flavors
- Can swap the 'Back' and 'Recent apps' buttons for left or right handed usage


Extra Options

Screenshot - takes the screenshot without having to press the combination of volume and power buttons
Volume - shows the volume dialog
Quick Settings - shows quick settings panel
Lock Screen - turns off the screen (standby mode)
Power Menu - shows power menu options

Note - These extra options will vary depending upon the Android version and the device.

Note - On some devices, clearing the recent apps or cache will exit the Android Navbar.

In such case, either

1. Disable Android Navbar from Accessibility Settings and enable it again.
or
2. Restart the device


Usage of Accessibility API

Android Navbar requires accessibility service for showing the navigation bar and extra options on screen using the default functions and features that are mentioned in 'Accessibility Service'.

Therefore, it's mandatory for Android Navbar to use the Accessibility Service API for accessing it's functions or methods as without accessibility service the app won't work.


Emergency Mode

1. Go to ---> Settings ---> Accessibility ---> Android Navbar
2. Enable/Disable Android Navbar
