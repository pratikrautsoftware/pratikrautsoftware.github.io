Result Analysis


 

 

Result Analysis

Version - 1.0

 

Result Analysis tool is created specifically for managing the result data where it currently exports the results from PDF file to Excel sheets subject-wise for reference or record purposes and also aims for performing analysis on result data by showcasing the data visually through graphs, charts and saving the required data into Word file.


Features


- User-friendly UI with easy-to-use interface.

- Exports result data from PDF file to Excel sheet at an ease.

- Reduces hours of manual work and gets it done within 10 minutes.

FAQ


Answer

 

1. Why does it show message with zero subject codes and produce errors despite verifying the submitted data with that of the PDF file?

Answer

If it still shows zero or no subject codes even after manual verification of the submitted data then it could be due to the PDF's format being different or the 'Result Analysis' not being executed as administrator.
